# Terminal Workflow Guide: Claude Code + Helix + Yazi

## Layout

```
┌──────────────┬──────────────┬────────────┐
│ Claude Code  │    Helix     │    Yazi    │
│   (左ペイン)  │  (中央ペイン)  │ (右ペイン)  │
│              │              │            │
│  AIと対話     │ ファイル/diff  │ ファイル構造 │
│  コード生成   │   閲覧        │  俯瞰      │
└──────────────┴──────────────┴────────────┘
```

Claude Code のフック発火時に自動で3ペインが作成される。
手動で開く場合は cmux でペインを追加して `hx` / `yazi` を起動。

## Helix 基本操作

IntelliJ との対応表。Helix はモーダルエディタ（Normal / Insert / Select モード）。

### モード切替

| キー | 動作 | IntelliJ相当 |
|------|------|-------------|
| `i` | Insert モード（編集） | そのまま入力 |
| `Esc` | Normal モードに戻る | Esc |
| `v` | Select モード（選択） | Shift+矢印 |

### ファイル操作

| キー | 動作 | IntelliJ相当 |
|------|------|-------------|
| `Space f` | ファイルピッカー（ファジー検索） | Cmd+P / Cmd+Shift+O |
| `Space F` | カレントファイルのディレクトリからピッカー | - |
| `Space b` | 開いているバッファ一覧 | Cmd+E (Recent Files) |
| `:open <path>` | ファイルを開く | File > Open |
| `:q` | 閉じる | Cmd+W |
| `:q!` | 保存せず閉じる | - |

### 検索

| キー | 動作 | IntelliJ相当 |
|------|------|-------------|
| `/` | ファイル内検索 | Cmd+F |
| `n` / `N` | 次/前の検索結果 | Cmd+G / Cmd+Shift+G |
| `Space /` | プロジェクト全体をgrep検索 | Cmd+Shift+F |
| `*` | カーソル下の単語を検索 | Cmd+F → 選択中の単語 |

### LSP ナビゲーション

| キー | 動作 | IntelliJ相当 |
|------|------|-------------|
| `gd` | 定義へジャンプ | Cmd+B / Cmd+Click |
| `gr` | 参照一覧 | Alt+F7 |
| `gi` | 実装へジャンプ | Cmd+Alt+B |
| `gy` | 型定義へジャンプ | Cmd+Shift+B |
| `Space s` | ファイル内シンボル検索 | Cmd+F12 (File Structure) |
| `Space S` | ワークスペースシンボル検索 | Cmd+Shift+O |
| `Space k` | ホバー情報（型、ドキュメント） | Ctrl+Q (Quick Doc) |
| `Space a` | コードアクション | Alt+Enter |
| `Space d` | 診断(エラー/警告)一覧 | Alt+1 (Problems) |
| `Space r` | リネーム | Shift+F6 |
| `]d` / `[d` | 次/前の診断 | F2 / Shift+F2 |

### 移動

| キー | 動作 |
|------|------|
| `gg` | ファイル先頭 |
| `ge` | ファイル末尾 |
| `:<行番号>` | 指定行へジャンプ (`:42`) |
| `Ctrl+u` / `Ctrl+d` | 半ページ上/下スクロール |
| `mm` | マッチする括弧へジャンプ |
| `Ctrl+o` / `Ctrl+i` | ジャンプ履歴 戻る/進む |

### 分割・複数バッファ

| キー | 動作 |
|------|------|
| `Ctrl+w v` | 縦分割 |
| `Ctrl+w s` | 横分割 |
| `Ctrl+w h/j/k/l` | 分割間移動 |
| `Ctrl+w q` | 分割を閉じる |

### diff 閲覧時 (delta)

delta は less で表示される。

| キー | 動作 |
|------|------|
| `j` / `k` | 1行 下/上 |
| `Space` | 1ページ下 |
| `b` | 1ページ上 |
| `/pattern` | 検索 |
| `q` | 閉じる |

## Yazi 基本操作

ファイルマネージャー。3カラム構成: 親ディレクトリ | カレント | プレビュー。

### ナビゲーション

| キー | 動作 | IntelliJ相当 |
|------|------|-------------|
| `j` / `k` | 上/下移動 | 矢印キー |
| `Enter` | **Helix pane でファイルを開く** | ダブルクリック |
| `l` / `Right` | ディレクトリに入る | 展開 |
| `h` / `Left` | 親ディレクトリへ | 折りたたみ |
| `gg` | 先頭へ | Home |
| `G` | 末尾へ | End |

### 検索・フィルタ

| キー | 動作 |
|------|------|
| `/` | ファイル名インクリメンタル検索 |
| `f` | フィルタ（マッチするファイルだけ表示） |
| `s` | ソート切替（名前/サイズ/更新日時） |

### ファイル操作

| キー | 動作 |
|------|------|
| `Space` | 選択トグル |
| `y` | コピー（ヤンク） |
| `x` | カット |
| `p` | ペースト |
| `d` | 削除（確認あり） |
| `r` | リネーム |
| `a` | 新規ファイル作成 |
| `A` | 新規ディレクトリ作成 |

### 表示

| キー | 動作 |
|------|------|
| `.` | hidden ファイル表示切替 |
| `z` | zoxide でディレクトリジャンプ |
| `~` | ホームディレクトリへ |
| `Tab` | タブ切替（複数ディレクトリ） |

## 自動連携の仕組み

### Claude Code → Helix (自動)

Claude Code がファイルを編集すると、フックスクリプト (`bin/claude-helix-bridge.sh`) が自動発火:

1. **Edit/Write ツール実行後**: `git diff` を取得し、中央ペインの Helix に delta (side-by-side diff) を表示
2. **セッション終了時**: 全変更の diff を delta で表示
3. **diff がない場合** (新規ファイルなど): Helix でファイルを直接開く

### Yazi → Helix (Enter キー)

Yazi でファイルを選んで Enter を押すと、`bin/yazi-open-in-helix.sh` が:

1. 中央ペインを自動検出
2. Helix が動いていれば `:open` で開く
3. shell プロンプトなら `hx` で起動

## よく使うワークフロー

### コードレビュー

1. Claude Code にコード変更を依頼
2. → 自動で中央ペインに diff が表示される
3. delta で diff を確認 (`j`/`k` でスクロール、`q` で閉じる)

### ファイル構造の把握

1. 右ペインの Yazi でプロジェクトを見渡す
2. 気になるファイルで `Enter` → Helix で開く
3. `gd` で定義ジャンプ、`gr` で参照確認

### 特定のコードを探す

- **ファイル名で**: Helix で `Space f` → ファジー検索
- **コード内容で**: Helix で `Space /` → grep 検索
- **シンボル名で**: Helix で `Space S` → ワークスペースシンボル検索

### git の変更確認

Helix 上で差分を見る:
- ガター (左端) に追加/変更/削除マークが表示される
- `]d` / `[d` で変更箇所間を移動
