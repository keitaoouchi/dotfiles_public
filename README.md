# Dotfiles

Personal development environment configuration for macOS (zsh).

## Prerequisites

1. **Homebrew**
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. **1Password** + **1Password CLI** (for git signing & secrets)
   ```bash
   brew install --cask 1password 1password-cli
   eval $(op signin)
   ```
   Create a vault named `Private` with:
   - Item `dotfiles`: fields `git_name`, `git_email`, `git_signing_key`
   - Item `secrets`: field `github_token`

## Installation

```bash
git clone https://github.com/keitaoouchi/dotfiles_public.git ~/.dotfiles_public
cd ~/.dotfiles_public
./setup.sh
```

## File Structure

```
~/.dotfiles_public/
├── setup.sh            # Run this to install everything
├── zshrc               # → ~/.zshrc
├── alias.sh            # → ~/.alias.sh
├── gitconfig           # → ~/.gitconfig  (includes ~/.gitconfig.local)
├── gitignore_global    # → ~/.gitignore_global
├── tmux.conf           # → ~/.tmux.conf
└── config/
    └── my.cnf          # → ~/.config/my.cnf
```

`~/.gitconfig.local` (git user info) is generated by `setup.sh` from 1Password and is **not** committed.

## Updating

```bash
cd ~/.dotfiles_public
git pull
./setup.sh
exec zsh
```
