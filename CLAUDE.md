# CLAUDE.md

This file provides guidance to AI agents working with this repository.

## Overview

macOS-only dotfiles for zsh. Symlink-based, no framework.

## File Structure

- **setup.sh** - Run once to install everything and create symlinks
- **zshrc** - Symlinked to `~/.zshrc`. Main zsh config (PATH, tools, completions)
- **alias.sh** - Symlinked to `~/.alias.sh`. Shell aliases
- **gitconfig** - Symlinked to `~/.gitconfig`. Includes `~/.gitconfig.local` for user info
- **gitignore_global** - Symlinked to `~/.gitignore_global`
- **tmux.conf** - Symlinked to `~/.tmux.conf`
- **config/my.cnf** - Symlinked to `~/.config/my.cnf`
- **config/helix/** - Symlinked to `~/.config/helix/`. Helix editor config (viewer-oriented, LSP)
- **config/yazi/** - Symlinked to `~/.config/yazi/`. Yazi file manager (IDE-like project tree)
- **bin/claude-helix-bridge.sh** - Claude Code hook: shows diffs/files in Helix pane via cmux
- **bin/yazi-open-in-helix.sh** - Yazi integration: opens selected file in Helix pane via cmux

## Secrets

- `~/.gitconfig.local` - Generated by `setup.sh` from 1Password (not in repo)

## How to Apply Changes

Edit a file in this repo, then the symlink means the change is live immediately.
Run `./setup.sh` again only if adding new symlinks or installing new packages.
